## Design Summary

优化 Kubernetes Agent WebUI 的整体页面布局，增加全局 Header，移除左 Nav 图标导航，让界面更协调、美观、简约大气。

**核心变更：**
1. 顶部新增全局 Header（Logo + 全局导航 + 主题切换）
2. 移除现有的左侧 60px 图标 Nav 栏
3. 主内容区随之调整为全屏宽度
4. 三栏布局（Nav + Sessions + Chat）变为两栏（Header + ChatView 内置 sessions panel）

---

## Alternatives Considered

### 方案 A：极简 Header
- **做法**：顶部 Header 只放 Logo/应用名 + 主题切换，不加导航
- **優點**：最简洁，改动最小
- **缺點**：导航入口消失，用户需要其他方式切换视图
- **為何未採用**：Nav 栏移除后需要替代导航方式

### 方案 B：功能型 Header（已選）
- **做法**：Header 内置全局导航标签（对话/集群/策略），主题切换按钮，以及 Logo/应用名
- **優點**：导航清晰，视觉统一，符合"美观简约大气"的设计目标
- **缺點**：Header 高度会增加（预计 48-56px），占用内容空间
- **為何採用**：满足用户"增加页眉"需求，同时解决左侧 Nav 图标占位问题

### 方案 C：保留 Nav 仅优化 Header
- **做法**：保留左侧 60px Nav，只在顶部加 Header
- **優點**：改动最小
- **缺點**：左侧图标 Nav 和顶部 Header 功能重复，界面不够简洁
- **為何未採用**：不符合"简约大气"的目标，冗余导航

---

## Agreed Approach

**方案 B：功能型 Header**

整体布局调整：
```
调整前：
┌──────────────────────────────────────┐
│ ☰ Nav │        main content         │
└──────────────────────────────────────┘

调整后：
┌──────────────────────────────────────┐
│ Logo │ 对话 │ 集群 │ 策略 │  Theme │ ⚙ │
├──────────────────────────────────────┤
│ Sessions │                            │
│ Panel   │     Chat/Messages          │
│ (240px) │                           │
└──────────────────────────────────────┘
```

**Header 设计（高度 48px，深色背景）**：
- 左侧：Logo 图标 + 应用名
- 中间：导航标签（对话/集群/策略），当前高亮
- 右侧：主题切换 + 设置按钮

**Sessions Panel 保留在 ChatView 内部**，宽度 240px，不影响现有功能。

---

## Key Decisions

1. **Header 高度 48px**，与当前 Nav 图标栏视觉平衡
2. **全局导航标签**替代左侧图标 Nav，点击切换视图
3. **Sessions Panel** 保持在 ChatView 内部（已实现），无需移动
4. **主题切换**移至 Header 右侧，与设置按钮并列
5. **深色主题下 Header 背景色**与当前 `--bg-sidebar` 一致（#010409）
6. **浅色主题下 Header 背景色**与 `--bg-elevated` 一致
7. **导航标签**使用 underline 指示当前视图，与现有 Tab 风格一致
